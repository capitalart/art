# Navigation QA Checklist

Use this checklist to manually verify navigation for both regular and admin users. Check each link on desktop and mobile sizes.

## Regular User
- [ ] Artwork Home link present
- [ ] Upload link present
- [ ] Gallery link present
- [ ] Finalised link present
- [ ] Exports link present
- [ ] Prompt Whisperer link present
- [ ] Docs dropdown lists all How-To pages
- [ ] About, Terms and Privacy links visible
- [ ] Every Docs page renders correctly
- [ ] No admin or upgrade links displayed

## Admin User
- [ ] All regular links visible
- [ ] Profile menu shows Account and Logout
- [ ] Admin link visible inside profile menu
- [ ] Upgrade link visible when feature flag enabled
- [ ] Account page accessible
- [ ] All documentation links accessible

Ensure every link loads without error on desktop and mobile layouts.
