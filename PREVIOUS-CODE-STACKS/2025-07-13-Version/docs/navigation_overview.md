# Navigation Overview

This document summarises the final structure of the Society6‑style top navigation bar. Regular users see only public modules, while admin tools appear in the profile menu.

- Artwork Home (logo link)
- Upload
- Gallery
- Finalised Gallery
- Exports
- Prompt Whisperer
- Docs dropdown with How‑To pages, FAQ, About, Terms and Privacy links
- Profile menu with Login/Logout
- Account, Upgrade and Admin options (admin only)
